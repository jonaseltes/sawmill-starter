Sawmill

A Modular Layout Builder for Hugo and Forestry.io

Sawmill is a simplistic Hugo theme built to take advantage of Forestry CMS’s Blocks feature.

This theme includes a few basic components to get started with Blocks right away and create dynamic, visually interesting layouts. Once you get the hang of Sawmill’s modular content strategy, you are encouraged to start creating your own components to fit the needs of your website.

Features

Built With Bulma

Bulma is a modern, responsive CSS framework with a flexbox-based grid system.

Customizable Branding

The theme’s primary color and logo are customizable from your config.toml file.

Made For Forestry

This theme is designed to work out-of-the-box with Forestry.io’s Blocks feature. Just copy the theme’s .forestry/front_matter/templates directory into the top level of your project, or get started even faster with the starter project.
